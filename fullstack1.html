Lipstick Searching
- Designed and developed a friendly user interface with JavaScript, HTML and CSS, allowing users to browse lipstick shades, access detailed product descriptions, and explore curated recommendations based on their preferences.
- Implemented a Node.js and Express.js backend architecture, integrating MongoDB to manage a comprehensive database of lipstick products, user profiles, and community posts.
- Built an interactive recommendation feature using user-provided data such as skin undertones, texture preferences, and occasions to deliver personalized lipstick suggestions.
- Enhanced user engagement by creating a community feature where users could share beauty tips, write reviews, and connect with like-minded individuals in a secure and scalable environment.
